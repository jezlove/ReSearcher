Downloading

ReSearcher provides the ability to download course materials directly from the Open University website. Follow the steps below to get started!

Note: the target directory for downloads can be configured in the application settings, see: Configuring, Locations, Downloads location. By default it is the user's Downloads directory.

Steps

The download process has four steps:

  1. Signing-in
  2. Checking for downloads
  3. Selecting files to download
  4. Downloading
