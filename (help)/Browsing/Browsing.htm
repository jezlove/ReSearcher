Selecting

To select a single file or directory simply: click on the row.

To select one or more additional files and/or directories: hold-down the CTRL key and then click on another row. Alternatively, you can use the ARROW UP / ARROW DOWN keys to move up or down and then the SPACE key to select.

To block-select multiple files and/or directories: hold-down the SHIFT key and then click on another row or use the ARROW UP / ARROW DOWN keys to select a range based on this and the initial selection.

Note: if you need more space to view: the form itself is resizable and scrollbars are also provided. Additionally, the table headings are also resizable.

Sorting

By default, the files are sorted in alphabetical order based on Name. To sort the files based on Type, Size (using numerical order), or Date (using chronological order): click on the respective table heading.

To alternate between ascending and descending order: click again on the already selected table heading. This will toggle from ascending to descending or from descending to ascending based on the current state.

Opening documents

To open any single selected file or directory, perform a double-click of that row.

To open multiple selected files or directories, click the Open... toolbar button or the ReSearcher > Open... menu item.

For more information, see: Opening documents.

Searching

To begin a search, first select any files and/or directories that you would like to search within. Then click the Search... toolbar button or the ReSearcher > Search... menu item. This will open the Search form, proceed to: Searching, Step 1. Specifying Criteria.

The Search form will be populated with the list of files that you selected and will also use the selected filter from the open Browse view tab.

Opening a new Browse view tab

To open a new Browse view to a directory on your machine: click on the Browse... toolbar button or the ReSearcher > Browse... menu item. This will open the Browse for folder dialogue box (shown in Figure 2).

Figure 2. Browse for folder dialogue box

Once you have selected the desired starting point folder: click on the OK button to close the dialogue box. This will open a new Browse view tab to the specified directory.

Or to cancel: click on the Cancel button.

Closing a Browse view tab

To close a Browse tab: select the Tab > Close option from the menu, as shown in Figure 2 below.

Figure 2. Main form with Browse View and Tab > Close menu item highlighted

Starting location

When ReSearcher starts-up a Browse view tab will be automatically opened to a pre-defined Starting location. This can be configured from the Settings form, see: Configuring, Locations, Starting location for more information.
