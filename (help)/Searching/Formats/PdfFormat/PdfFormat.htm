PDF format

PDF (Portable Document Format) is a popular document format developed by Adobe.

ReSearcher supports searching within PDF files and can narrow down searches to the page-level.

Format description

For more information on the specifics of the PDF format, consult the ISO standard: https://www.iso.org/standard/63534.html.

Viewing

For more information on how to configure a viewer for documents of this format, see: Configuring, PDF viewer.
