EPUB format

EPUB (also variously styled as ePUB or ePub) is a popular document format originally developed by the IDPF (International Digital Publishing Forum), prior to their merger with the W3C (World Wide Web Consortium).

ReSearcher supports searching within EPUB files and even permits the use of XPath queries to narrow down searches. This is because the format is essentially a Zip archive of XHTML documents (navigable by XPath), along with some other supporting files. For more information on searching with XPath queries, see XPath expressions.

Note: presently, only the XHTML files within an EPUB are searchable.

Format description

For more information on the specifics of the EPUB format, consult the W3C standard: https://www.w3.org/publishing/epub3/epub-spec.html.

XHTML

For more information on XHTML, see: the W3C standard https://www.w3.org/TR/xhtml1/ or for a brief introduction see: https://www.w3schools.com/html/html_xhtml.asp.
