Word format

'Word format' is a term used by ReSearcher to actually describe two distinct formats:

  • Binary format (Word 97â€“2007)

    Most typically with a '.doc' extension (seen less often these days)

  • WordprocessingML (Office Open XML format)

    Most typically with a '.docx' extension

Both formats are developed by Microsoft (although, the latter done with outside collaboration).

ReSearcher supports searching within both varieties. However, for the former (.doc) Microsoft Word must be installed in order to search them. Whereas, for the latter (.docx) an installation of Microsoft Word is not required.

Format description

Standards have been published for both formats. However, the legacy Binary format is no longer being actively developed.

Binary format

A formal description of this standard is available to download as a PDF: https://download.microsoft.com/download/0/B/E/0BE8BDD7-E5E8-422A-ABFD-4342ED7AD886/Word97-2007BinaryFileFormat(doc)Specification.pdf [PDF].

Office Open XML format

Essentially, an (OOXML) Word document is a Zip archive containing XML files (combined with other supporting files). However, unlike EPUB documents, ReSearcher does not presently support using XPath queries when searching them.

For more detailed information, see the ECMA standard: http://www.ecma-international.org/publications/standards/Ecma-376.htm.
