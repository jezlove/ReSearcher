Step 1. Specifying criteria

The Search form is shown in Figure 1 below.

Figure 1. Search form

Editing selected paths and selected filter

The input box below the Path(s) and filter: heading (shown in Figure 1), is for editing the target search files and/or directories to be searched. The drop-down box to the right is for editing the filter to be used for any directories in that list.

Note: if you choose to make any edits to this list, a single line should be used for each path.

Normally, the list of selected paths and selected filter should already be populated prior to the displaying of the this window. This information comes from any selected files and directories being viewed in the active Browse view of the Main form.

If no Browse view tab was open then this list will be empty, but can be entered manually on this screen. It is recommended in this case, to cancel and browse for files and directories using the Browse view, see: Browsing.

Entering XPath expressions

The input box below the XPath (within EPUB only): heading (shown in Figure 1), is specifically for entering XPath expressions only.

The XPath expression is used only when searching through EPUB documents, to locate a specific node (part of the document) to search within.

For more information about writing XPath expressions, see: XPath expressions.

Entering Simple, Boolean, or Regex expressions

The tabs Simple, Boolean, and Regex (shown in Figure 1) are for entering Simple, Boolean, or Regular expressions respectively. For more information about these different types of expressions, see: Expressions.

Regular expression options

Under the Regex tab (for entering regular expressions) there are also additional checkboxes for selecting options that will affect the way that the regular expression matches data. The single character next to them is a common abbreviation for that option that some users might be familiar with.

By default, the ignore case and ignore whitespace options are selected.

For more information about regular expressions, see: Regular expressions.

Searching

When you have finished entering your required search criteria, click on the Search... button. This window will close and ReSearcher will begin conducting a new search with this criteria.

Note: this may take some time to complete.

Once ReSearcher has collected all of the results fitting your criteria, the Results view will appear to display this information. Proceed to Step 2.

Note: this search will be saved to your Search history. For more information about this, see: History.

Cancelling

If you would like to cancel: click on the Cancel button. This will abandon the search process and close this window.

Loading a past search

To load a past search into the Search form: click the Load... button and the Search history form will appear. See: History, Retrieving past search criteria for more information.
