Simple expressions

Naturally, simple expressions are the simplest type of search term. They are exact character for character matches, so for example:

dog

Matches:

dog

Case insensitivity

Simple expressions are case insensitive, so:

cat

Matches all of:

cat, Cat, cAt, CAt, caT, CaT, cAT, CAT

Whole words only

Simple expressions search for whole words only, so:

pet

Will not match any of:

petulant, carpet, trumpeter, etc.

Whitespace

Simple expression searches ignore whitespace (spaces, tabs, line endings, etc.) in both the search expression and within search results. So both:

Guinea pig

and

Guinea     pig

Will match both:

Guinea pig and Guinea     pig

Where underlining is used above for illustration of any whitespace.

Note: not to be confused with the '_' character, see also: Other characters below.

Other characters

Simple expressions consider letters and digits (of any spoken language) to be part of a word. Also acknowledged are:-

  • ''' (Unicode/ASCII code 39 - single-quote/apostrophe)
  • '-' (Unicode/ASCII code 45 - hyphen/minus/dash)
  • '_' (Unicode/ASCII code 95 - underscore/understrike/underbar/low-line)

All other characters are ignored.

Note: for more information about Unicode, check out: https://www.unicode.org/.
