Boolean expressions

Boolean expressions are very similar to Simple expressions, however, they offer some more control over results. This is done by the boolean keywords 'AND' and 'OR', so:

dog bed

and

dog AND bed

Both match:

dog bed

However:

dog OR bed

Matches either:

dog or bed (or dog bed, but as two separate matches)

This is useful for filtering and finding more results.

Case insensitivity

Boolean expressions are, for the most part, case insensitive (but not keywords, see Keywords and case insensitivity below), so:

cat

Matches all of:

cat, Cat, cAt, CAt, caT, CaT, cAT, CAT

Keywords and case insensitivity

The boolean keywords 'AND' and 'OR' are case sensitive, so:

cat and mouse

Matches:

cat and mouse

But not:

cat mouse

Whereas cat AND mouse would.

Whole words only

Boolean expressions search for whole words only, so:

pet

Will not match any of:

petulant, carpet, trumpeter, etc.

Whitespace

Boolean expression searches ignore whitespace (spaces, tabs, line endings, etc.) in both the search expression and within search results. So both:

Guinea pig

and

Guinea     pig

Will match both:

Guinea pig and Guinea     pig

Where underlining is used above for illustration of any whitespace.

Note: not to be confused with the '_' character, see also: Other characters below.

Other characters

Boolean expressions consider letters and digits (of any spoken language) to be part of a word. Also acknowledged are:-

  • ''' (Unicode/ASCII code 39 - single-quote/apostrophe)
  • '-' (Unicode/ASCII code 45 - hyphen/minus/dash)
  • '_' (Unicode/ASCII code 95 - underscore/understrike/underbar/low-line)

All other characters are ignored.

Note: for more information about Unicode, check out: https://www.unicode.org/.
