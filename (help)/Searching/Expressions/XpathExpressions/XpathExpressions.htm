XPath expressions

XPath (or XML Path Language) is a language for querying xml documents. ReSearcher supports EPUB, a popular document format but one which is essentially a Zip archive of XHTML documents (combined with other supporting files). XHTML is a subset of XML, this enables the use of XPath queries to select nodes (parts of) of an XHTML document to search within. The use for this feature would be if you wanted to search only in tables (.//table) or only within headings (.//h1|.//h2|.//h3|.//h4|.//h5|.//h6).

Because this is a broad and complicated topic, I will not be elaborating in any greater detail on how to use XPath expressions as it has already been better explained online. See: https://www.w3schools.com/xml/xpath_intro.asp for a brief introduction to XPath. And also, for more information about XHTML, see: the W3C standard https://www.w3.org/TR/xhtml1/ or for a brief introduction see: https://www.w3schools.com/html/html_xhtml.asp.

Note: when browsing external sources for information regarding XPath expressions make sure that you only use constructs that are supported by XPath version 1.0 (and the .Net Framework XPath implementation). See .Net XPath implementation below for more details.

.Net XPath implementation

XPath is a format (among many) created by the W3C (World Wide Web Consortium) but different vendors have created implementations of it. ReSearcher uses the XPath version 1.0 implementation provided by the .Net Framework. More information on the specifics of this implementation can be found at: https://docs.microsoft.com/en-us/previous-versions/dotnet/netframework-3.5/ms256115(v=vs.90). Additionally, to find out more information about XPath version 1.0, see: https://www.w3.org/TR/1999/REC-xpath-19991116/. Lastly, to compare XPath version 1.0 to other versions, see: https://www.w3.org/TR/xpath/.

Word (Office Open XML format)

Another popular document format that uses XML within a Zip archive is Office Open XML, in particularly: Word (WordprocessingML). However, searching with XPath expressions within this format is not presently supported.

For more information on the Word format, see: Word format.
