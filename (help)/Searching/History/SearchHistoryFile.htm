Search history file

Caution: this topic is intended only for advanced users.

This topic is provided solely so that you may backup this file for use again with this version of ReSearcher.

Your search history is saved to the file: ReSearcher.json, located in your %userprofile% directory. The file is saved in JSON format, see the Format section below for more details.

Caution: making changes directly to the ReSearcher.json file could make the whole file unreadable if a mistake is made.

Editing by hand is not recommended.

For information on how to retrieve past search criteria normally, see Retrieving past search criteria.

Format

Caution: this format is subject to change at any time and is accurate only for this version of ReSearcher.

If you are unfamiliar with the JSON format, see: https://www.json.org/.

ReSearcher uses the Newtonsoft JSON implementation to read and write to this file. For more information on the Newtonsoft JSON implementation, see: https://www.newtonsoft.com/json.

The format for a record is:

[
    {
         "paths": [
               String,
              ...
         ],
         "filter": String,
         "xpath": String,
         "searchType": String,
         "pattern": String,
         "regexOptions": Number
    },
    ...
]

Where String and Number represent a JSON String and Number respectively.

The number for regexOptions is a bitfield representation of the .Net System.Text.RegularExpressions.RegexOptions enumeration (https://docs.microsoft.com/en-us/dotnet/api/system.text.regularexpressions.regexoptions).
