Configuring

Configuration of ReSearcher is done by the Settings form. To access: click on the ReSearcher > Settings menu item (shown in Figure 1).

Figure 1. Settings menu item

This will launch the Settings form (shown in Figure 2).

Figure 2. Settings form

From this window all settings can be viewed and edited (except for your OU username and password, see Username and Password below).

Note: settings are updated automatically upon editing, there is no need to 'apply' changes for them to take affect.

When done making changes: click the Done button to close the window.

Username and password

Your username and password are entered into the Sign-in form only (see Downloading, Step 1. Signing-in). If you select the Remember me checkbox then your username and your password (an encrypted version) will be saved into the configuration file.

Search history

Note: your search history is not saved as part of the configuration file.

For more information on Search history, see: Searching, History.
