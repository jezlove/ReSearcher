Downloads location

The Downloads location is the directory where all new content will be saved upon downloading. For more information about downloading new content, see: Downloading.

Default location

The default Downloads location is the user's Downloads directory. Typically: %userprofile%\Downloads\.

Editing

Shown in Figure 1 is the panel for editing the Downloads location.

Figure 1. Settings form with Downloads location (circled)

To type in a location: click on the input box below the Download to: heading (shown in Figure 1). Either begin typing or paste in the path from the clipboard (perform a right-click and select Paste or use the keyboard shortcut CTRL + V).

Browsing for folder

To browse for a directory on your machine: click on the ... button to the right of the input box to open the Browse for folder dialogue box (shown in Figure 2).

Figure 2. Browse for folder dialogue box

Once you have selected the desired folder: click on the OK button to close the dialogue box.

Or to cancel (and leave the Downloads location unchanged): click on the Cancel button.
