Viewers

From the Browse view, any file can be launched and is done so using the program associated with the file extension on your machine (see: Browsing, Opening documents).

However, to launch search results a specific external viewer application needs to be configured. This is because some applications allow for the passing of parameters by ReSearcher to tell the application where to open the specifically open the document to. Which is very useful for getting to results faster.

ReSearcher supports configuring three viewers (one for each searchable format):

  • EPUB viewer

    See also: EPUB format.

  • PDF viewer

    See also: PDF format.

  • Word viewer

    See also: Word format.
