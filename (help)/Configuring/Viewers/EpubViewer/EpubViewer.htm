EPUB viewer

The EPUB Settings tab is displayed in Figure 1 below:

Figure 1. Settings form with EPUB tab (circled)

Fields

  • Open with: specifies the program file location.

  • To file: specifies the argument format pattern for simply opening a file (but allowing for the user to specify additional parameters to be passed).

  • To section in file: specifies the argument format pattern for opening a file to a specific section (specific XHTML topic within the EPUB file).

Defaults

  • Open with: ReSearcher checks to see if a known application is installed (and attempts to configure for that application). Otherwise, the field is left blank for the user to enter.

  • To file: "{0}"

    Where {0} is a placeholder for the file path of the document to be opened.

  • To section in file: "{0}"

    Where {0} is a placeholder for the file path of the document to be opened.

    The placeholder {1} (for a topic title) is not included unless it is known where in the pattern it is acceptable.

Calibre 2

The Calibre 2 ebook-viewer.exe application provides the following command-line help: https://manual.calibre-ebook.com/generated/en/ebook-viewer.html.

The option --open-at (https://manual.calibre-ebook.com/generated/en/ebook-viewer.html#cmdoption-ebook-viewer-open-at) can be used to open to a section in the file. The pattern for this would be:

--open-at toc:"{1}" "{0}"
